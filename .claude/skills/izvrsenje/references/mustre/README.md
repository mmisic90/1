# Korpus mustri — predlozi za izvršenje (stvarni akti kancelarije)

28 stvarnih .docx akata (predlozi za izvršenje na osnovu verodostojne i izvršne
isprave, odgovori na prigovor, podnesak — protivljenje odlaganju) + 2 skenirana PDF-a:

- `2_FORMULACIJE_iz_mustri.pdf` — destilovane formulacije iz mustri (sken, 11 str;
  čitati vizuelno po stranicama)
- `SKILL_izvrsenje.pdf` — skill u PDF obliku (sken, 6 str)

## Kako koristiti
Pri izradi predloga za izvršenje: otvoriti 1–2 najbliže mustre po tipu
(verodostojna isprava = računi/fakture; izvršna isprava = presuda) i klijentu
(Vukovic 2015, Armex, Tape NS, Agrobiochem, Energy LTD, Ab Pro Ing…) i pratiti
strukturu, petit i troškovnik. Slojevite kamate i naming po stil-pisanja pravilima.

Napomena: dokumenti sadrže stvarne generalije stranaka — repo mora ostati PRIVATAN.
