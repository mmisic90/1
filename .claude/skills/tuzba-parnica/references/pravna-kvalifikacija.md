# Правна квалификација — стандардне формулације

## Уводна реченица правне квалификације
```
Имајући у виду све напред наведено, предлажем да суд после извођења предложених доказа на главној расправи донесе следећу
```
(Ако нема потребе за главном расправом, без „после извођења предложених доказа на главној расправи".)

## Позивање на законске одредбе — примери

### Пробој правне личности (чл. 18 ЗПД)
```
Чланом 18 ставом 1 ЗПД-а одређено је: „Командитор, члан друштва са ограниченом одговорношћу и акционар, као и законски заступник тог лица, ако је оно неспособно физичко лице, који злоупотреби правило о ограниченој одговорности, одговара за обавезе друштва."

Ставом 2 истог члана, одређено је: „Сматраће се да постоји злоупотреба из става 1 овог члана нарочито ако то лице: ... (4) ради стицања користи за себе или трећа лица умањи имовину друштва, иако је знало или морало знати да друштво неће моћи да извршава своје обавезе."
```

### Накнада штете (чл. 154, 185 ЗОО)
```
Одредбом чл. 154 ст. 1 ЗОО-а прописано је да ко другоме проузрокује штету дужан је надокнадити је, уколико не докаже да је штета настала без његове кривице.

У складу са чл. 185 ст. 1 ЗОО-а, накнада обухвата обичну штету и измаклу корист.
```

### Ништавост уговора (чл. 103 ЗОО)
```
Одредбом чл. 103 ст. 1 ЗОО-а прописано је да је уговор који је противан принудним прописима, јавном поретку или добрим обичајима ништав, ако циљ повређеног правила не упућује на неку другу санкцију или ако законом у одређеном случају није шта друго предвиђено.
```

### Паулијанска тужба (чл. 280 ЗОО)
```
Одредбом чл. 280 ст. 1 ЗОО-а прописано је да поверилац чије је потраживање доспело за исплату може побијати правне радње свога дужника предузете на штету повериоца.
```

### Закашњење са испуњењем (чл. 324, 326 ЗОО)
```
Одредбом чл. 324 ЗОО-а прописано је да је дужник у доцњи ако не испуни обавезу у року одређеном за испуњење.
```

### Законска затезна камата (ЗОСЗК)
```
Сходно одредби чл. 3 Закона о затезној камати, на износ неисплаћеног новчаног потраживања поверилац има право на затезну камату почев од дана доспелости обавезе.
```

## Завршна реченица пред петитом — варијанте

**Са главном расправом:**
```
Сходно свему напред наведеном, предлажем да суд после извођења предложених доказа на главној расправи донесе следећу
```

**Без главне расправе (нпр. платни налог):**
```
Сходно свему напред наведеном, предлажем да суд донесе следећу
```

**Са предлогом за привремену меру:**
```
Сходно свему напред наведеном, а имајући у виду постојање потраживања и опасности да ће се без привремене мере тужилац довести у немогућност наплате, предлажем да суд донесе следећу
```
