# Шаблон 02: Тужба са предлогом за привремену меру ⭐

## Када се користи
**Зломат стандард** Мишићевог стила — тужба која истовремено садржи и **предлог за одређивање привремене мере** из чл. 447-449 ЗИО. Користи се када постoji:

- **Опасност по потраживање** — тужени може располагати имовином пре окончања спора
- **Опасност по извршење** — тужени ликвидиран, отуђује имовину
- **Обезбеђење нефинансијских потраживања** — давања, чињења, нечињења, трпљења

**Законски основ:**
- **ЗПП** — чл. 192-212 (тужба)
- **ЗИО чл. 447** (привремена мера пре/у току поступка)
- **ЗИО чл. 449** (услови за одређивање привремене мере)

**Рок:** општи за тужбу + предлог за привр. меру се може поднети **било кад пре окончања поступка**.

**Надлежност:** **исти суд** који решава по тужби.

## Терминологија
- **ТУЖИОЦИ** (множина ако је више лица) или **ТУЖИЛАЦ**
- **ТУЖЕНИ 1, 2, 3. реда** (обавезно оzнака реда)
- У делу о привр. мери: **ПРЕДЛАГАЧ (поверилац)** и **ПРОТИВНИК ПРЕДЛАГАЧА (дужник)** — алтернативна терминологија, често се мешajу

## Default потпис
**Милан** (сам) за стандардну тужбу, али за **велике парнице преко 10М РСД — Милан + Ђорђе**.

## Тарифа
- **Тужба + предлог за привр. меру**: тарифа тужбе **+ увећање 50-100%** за предлог за привр. меру
- За спор 19.9М РСД (Kalentić): **56.250 РСД + 40.000 РСД = 96.250 РСД** (оквирно)

## Кључне напомене при писању

### 1. Структура — ДВА петита у једном акту

Специфичност овог шаблона: **два петита**:
1. **ПРЕСУДА** — по тужбеном захтеву (главни предмет)
2. **РЕШЕЊЕ** — по предлогу за привремeну меру

Оба петита су у истом акту, раздвojенa наслом "Р Е Ш Е Њ Е" (или "РЕШЕЊЕ О ПРИВРЕМЕНОЈ МЕРИ").

### 2. Цитирање ЗИО чл. 447 и 449 — ОБАВЕЗНО

Из Kalentić модела — **увек** експресно цитирати ове члaнове у курзиву са ,,...". Суд се понаса да ли је предлог адекватно правно основан.

### 3. Три услова за привремеmu меру (чл. 449 ЗИО)

**Сва три** услова морају бити демонстрирана:
1. **Вероватноћа потраживања** — из доказа
2. **Опасност по извршење** — финансијски извештаji, активности туженог
3. **Непоправива штета** или немогућност извршења у натури

**Ако нема било који — предлог се одбацује.**

### 4. CAPS емфазе у доказивању опасности

Из Kalentić — CAPS фразе:
```
КЊИГОВОДСТВЕНА ВРЕДНОСТ УКУПНЕ ИМОВИНЕ ТУЖЕНОГ 2. РЕДА...
ДРУШТВО ЈЕ ПОСЛОВАЛО ТОКОМ 2023. ГОДИНЕ ,,У МИНУСУ"...
АПСОЛУТНО учинили вероватним...
```

Ово су **кључне тврдње** — без њих суд неће видети опасност као стварну.

### 5. Именовање јавног извршитеља

У петиту за привр. меру — експресно се именује извршитељ:
```
ИМЕНУЈЕ се за спровођење ове привремене мере јавни извршитељ 
Жељко Кесић, са седиштем у Зрењанину, ул. Краља Александра 
Карађорђевића 15/III.
```

**Милан користи Жељка Кесића за Зрењанин** (према меморији) — проверити пре писања да ли је одговарајући за место туженог.

### 6. Рок трајања привр. мере (чл. 449 ст. 7 ЗИО)

**Највиши рок:**
- Ако је дужник физичко лице или правно лице → **1 година** од дана одређивања
- Ако је у питању парнични поступак који се води → **до правноснажности пресуде**

Обавезно у петиту:
```
Мера важи до правноснажности пресуде у овом поступку, а 
најдуже годину дана од дана доношења овог решења (у смислу 
чл. 449 ст. 7 ЗИО).
```

### 7. Обезбеђење предлагача (чл. 450 ЗИО)

Суд може тражити да предлагач **положи обезбеђење** (јемство) ако:
- Је одређивање привремене мере ризично за противника предлагача
- Постоји могућност прекомерне штете

**Формулација за нудити обезбеђење:**
```
Тужиоци предлажу да суд, уколико сматра да је обезбеђење 
неопходно, одреди обезбеђење у разумном износу (нпр. 
10% од вредности спора), а тужиоци су спремни да исто 
положе.
```

### 8. Шта НИКАД у тужби са привременом мером

- **Привремeна мера без конкретизованог предмета** (нпр. "забрана свега")
- **Тврдња о опасности без доказа** — обавезно финансијски извештаjи, активности
- **Без оснивачког акта или уговора** — стварноправни захтеви губе смисао
- **Задржавање привр. мере "заувек"** — увек ограничити трajање

- [ ] 3 примерка + прилози?
- [ ] Увод: "Као пуномоћник... подносим тужбу"?
- [ ] Чиnjенично стање хронолошки са **ДОКАЗ блоковима**?
- [ ] **А/Б/Ц структура** у чињеничном стању ако је сложено?
- [ ] Правна квалификација са цитатима у курзиву?
- [ ] **Секција за привр. меру** са цитатом **чл. 447 и чл. 449 ЗИО**?
- [ ] **Три услова** за привр. меру демонстрирана?
- [ ] CAPS емфазе (финансијски индикатори опасности)?
- [ ] Именован **јавни извршитељ**?
- [ ] **Рок трајања** привр. мере?
- [ ] РЕЗИМЕ?
- [ ] **Двоструки петит**: ПРЕСУДА + РЕШЕЊЕ?
- [ ] Потпис (Милан сам или оба за велики спор)?
- [ ] Трошковник (тужба + увећање за привр. меру)?
