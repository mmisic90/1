# Граф зависности аргумената + Протокол „Једна ствар"

## ЗАШТО ГРАФ ЗАВИСНОСТИ

Адвокат који има 5 аргумената мисли да је јак. Али ако сви
зависе од једне чињенице — и та чињеница падне — нема ништа.

Граф зависности показује:
- Који аргументи су НЕЗАВИСНИ (стоје сами)
- Који ЗАВИСЕ од других (ако X падне, Y такође пада)
- Која чињеница је ТЕМЕЉ за највише аргумената (критична тачка)

## КАКО НАПРАВИТИ ГРАФ

### Корак 1: Листај аргументе
```
АРГ #1: Битна повреда — неоцењен доказ (чл. 438)
АРГ #2: In dubio pro reo (чл. 16 ЗКП)
АРГ #3: Погрешна квалификација (чл. 439)
АРГ #4: Неразумљива изрека (чл. 438 ст. 1 тач. 11)
АРГ #5: Прекорачење казне
```

### Корак 2: За сваки пар — питај „Ако X падне, да ли Y пада?"
```
#1 пада → #2 пада? НЕ (различит основ) → НЕЗАВИСНИ
#1 пада → #3 пада? НЕ → НЕЗАВИСНИ
#1 пада → #4 пада? ДЕЛИМИЧНО (оба о образложењу) → СЛАБА ВЕЗА
#2 пада → #5 пада? НЕ → НЕЗАВИСНИ
#3 пада → #5 пада? ДА (ако је квалификација тачна, казна је у оквиру) → ЗАВИСАН
```

### Корак 3: Нацртај граф
```
    АРГ #1 (5/5) ──слабо──→ АРГ #4 (3/5)
    [НЕЗАВИСАН]              [ПОЛУЗАВИСАН]

    АРГ #2 (4/5)
    [НЕЗАВИСАН]

    АРГ #3 (4/5) ──јако──→ АРГ #5 (3/5)
    [НЕЗАВИСАН]              [ЗАВИСАН]
```

### Корак 4: Анализирај граф
- **3 независна аргумента** (#1, #2, #3) → ДОБРО. Чак и ако 2 падну,
  1 стоји.
- **Критична тачка:** Ниједна. Нема чињенице од које зависи СВЕ. ДОБРО.
- **Слабост:** Ако #3 падне, #5 аутоматски пада → не ослањај се на #5
  као самосталан аргумент.

### ЛОША СИТУАЦИЈА (пример):
```
    ЧИЊЕНИЦА: „Сведок К. видео окривљеног"
        ↓ ↓ ↓ ↓
    #1  #2  #3  #4  (сви зависе од овог сведока)
    #5 ← (зависи од #2)
```
Ако сведок К. промени исказ или буде дискредитован →
сва 4 аргумента + #5 падају. Ово је КАТАСТРОФАЛНО.
**Решење:** МОРАШ пронаћи барем 1 аргумент који НЕ ЗАВИСИ
од сведока К.

## ПРОТОКОЛ „ЈЕДНА СТВАР"

У сваком предмету постоји ЈЕДНА чињеница или ЈЕДАН правни
институт који тежи више од свега осталог. Нађи га.

### Четири питања за идентификацију:

**1. ФИЛТЕР ЈЕДНЕ РЕЧЕНИЦЕ:**
„Ако бих имао само 10 секунди пред судијом — шта бих рекао?"
Одговор на ово = вероватно „Једна ствар".

**2. ФИЛТЕР ЕЛИМИНАЦИЈЕ:**
„Ако избришем све аргументе осим једног — који задржавам?"
Онај који задржаваш = „Једна ствар".

**3. ФИЛТЕР НЕОДБРАЊИВОСТИ:**
„На шта противна страна НЕ МОЖЕ да одговори?"
Ако постоји такав аргумент — то је „Једна ствар".

**4. ФИЛТЕР ПРОМЕНЕ КОНТЕКСТА:**
„Која чињеница, ако се истакне, МЕЊА ЦЕЛУ СЛИКУ?"
Нпр.: Вештак дао алтернативно објашњење повреда.
Ово мења контекст од „да ли је окривљени ударио"
у „да ли УОПШТЕ знамо шта се десило".

### Где „Једна ствар" иде у документу:

**ПРВА.** Не чувај је за крај. Не закопавај је на страни 8.

Судије читају хиљаде поднесака. Ако „Једна ствар" није у
првих 2-3 пасуса образложења — можда је судија неће видети.

Структура документа:
```
1. Уводна формулација (стандардна)
2. „ЈЕДНА СТВАР" — најјачи аргумент, ОДМАХ
3. Остали аргументи (рангирани по снази)
4. Предлог суду
```

## АРХИТЕКТУРА АРГУМЕНТАЦИЈЕ

### Piramidalna struktura (за јаке случајeve)
```
         „ЈЕДНА СТВАР"
        /      |       \
   АРГ #2   АРГ #3   АРГ #4
      |        |
   под-арг   под-арг
```
Најјачи на врху. Сваки следећи ДОПУЊУЈЕ претходни.
Ако судија прочита само прва 2 пасуса — добио је суштину.

### Shotgun struktura (за слабе случајeve)
```
АРГ #1 (независан)
АРГ #2 (независан)
АРГ #3 (независан)
АРГ #4 (независан)
```
Кад ниједан аргумент није довољно јак сам — штити се количином
НЕЗАВИСНИХ аргумената. Ако судија одбије 3 од 4 — још увек
имаш 1.

### Cascade struktura (за процесне предмете)
```
АРГ #1: Битна повреда (чл. 438)
  → Ако суд ово не усвоји:
АРГ #2: Повреда крив. закона (чл. 439)
  → Ако ни ово:
АРГ #3: Чињенично стање (чл. 440)
  → Ако ни ово:
АРГ #4: Одлука о санкцији
```
Свака следећа линија је „план Б" за претходну. Покриваш
СВЕ основе за жалбу систематски.

## ТЕСТИРАЊЕ СНАГЕ АРГУМЕНТА

За сваки аргумент, одговори на 5 питања:

| Питање | Одговор | Бодови |
|---|---|---|
| Да ли је чињенична основа ✅П1? | да/не | +2/0 |
| Да ли правна норма директно покрива? | да/делимично/не | +2/+1/0 |
| Да ли има судску праксу? | да/не | +1/0 |
| Да ли је независан? | да/делимично/не | +2/+1/0 |
| Да ли противник може одговорити? | не/делимично/да | +2/+1/0 |

**Скала:** 8-10 = ■■■■■ (5/5), 6-7 = ■■■■□ (4/5),
4-5 = ■■■□□ (3/5), 2-3 = ■■□□□ (2/5), 0-1 = ■□□□□ (1/5)
